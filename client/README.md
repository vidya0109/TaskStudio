# Frontend Starter

Production-ready frontend scaffold with React, MUI, Jest, and Playwright.

## Run locally

1. Install dependencies:
   - `npm install`
2. Start dev server:
   - `npm run dev`
3. Run unit tests:
   - `npm run test`
4. Run e2e tests:
   - `npm run e2e`

## Structure

- `src/app`: app shell, routes, global providers
- `src/features`: feature-level components and pages
- `src/components`: shared presentational components
- `src/services`: API clients and domain service modules
- `tests/unit`: Jest unit tests
- `tests/e2e`: Playwright smoke/e2e tests
